# Advanced Marketing Models on R '24

## Overview
- 2 acts: first act is dedicated to basics, second act is dedicated to advanced topics
- Homework assignments after each seminar, quizes during seminars
- Grading scheme: 0.25 * Class Quizzes + 0.25 * Class Practice Cases + 0.25 * Midterm Exam + 0.25 * Final Exam

## Seminar 1: Basics
- introduction, course goals and organizational aspects
- briefly on 'how it may be helpful for business' and data-driven decision-making
- sampling (techniques and difference with population)
- understanding your data (types, descriptive statistics, distributions)
- experiment design and statistical significance
- R environment setup
- Homework 1: EDA for given dataset (summary + descriptive statistics)

## Seminar 2: Introduction to R Syntax and Data Preprocessing
- R base syntax (objects, variables, functions, loops, conditionals, lists, data frames, matrices, vectors, etc.)
- useful external libraries overview (dplyr, tidyr, ggplot2, etc.)
- data preprocessing principles (objects / NULLs and outliers)
- Quiz 1: Data Preprocessing
- Homework 2: EDA for given dataset for given dataset (data cleaning + descriptive statistics + plots)

## Seminar 3: Statistics Fundamentals
- how to run a statistical test in R (hypothesis testing basics) and how to interpret the results
- assumptions about data (normality, independence, homoscedasticity) and how to check them
- parametric and non-parametric tests for group comparisons (chi-square test, T-test, ANOVA, Mann-Whitney U test, Wilcoxon test, etc.)
- Quiz 2: Statistics Fundamentals
- Homework 3: Perform hypothesis testing for given dataset, write functions to perform statistical tests

## Seminar 4: Correlation and Regression
- linear models applications
- correlation: types (Pearson, Spearman, Kendall), assumptions, interpretation of the results, multicollinearity
- types of regression (linear, logistic, hierarchical, GLM, Bayesian, etc.)
- specifics for Linear, Logistic, Hierarchical, GLM, Bayesian models (what for, when applicable, useful details)
- Midterm Exam: assess distribution, run groups comparisons, build correlation and regression models for given dataset, interpret the results for each model run

## Seminar 5: Complexity Reduction, 
- normalising data and reducing its' complexity (scaling / normalisation / PCA), motivation for it
- EFA (Exploratory Factor Analysis) and MDS (Multidimensional Scaling)
- unsupervised methods (Clustering / Classification)
- Quiz 3: Dimensionality Reduction, EFA, MDS, Clustering, Classification
- Homework 4: perform complexity reduction techniques for given dataset (PCA, MDS), write functions to do a classification and clustering

## Seminar 6: Choice Modeling
- SEM (Structural Equation Modeling)
- CFA (Confirmatory Factor Analysis)
- Association Rules (Market Basket Analysis)
- Choice Modeling
- Recap of previously learned statistical tests and models
- Final Exam: Quiz on all topics + practical tasks on writing models to compare groups, forecast a linear variable, cluster and classify user groups (2 given datasets)
